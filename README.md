# Folo — Invest in Indian startups from ₹1

A discovery-first equity crowdfunding platform. Follow startups you believe in. Invest when they raise.

## Stack
- Next.js 16 (App Router)
- TypeScript
- Lucide React icons

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000
